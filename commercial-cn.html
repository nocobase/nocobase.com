商业版

为了满足用户的高级需求，同时也为了获得收入以保证 NocoBase 的健康发展，我们在开源版本的基础上提供了商业化的版本。 请注意，这并不是一个独立于开源版本的产品，正是得益于 NocoBase 的插件化架构，商业版本的高级功能全部以插件的方式提供。 这意味着开源版本永远是我们的基石，你可以放心使用。

商业版本分为 授权、服务、插件 三部分，你可以按需选择任何部分。

许可证 介绍 适用场景

插件

以下是商业插件，它们可以在开源版本基础上，极大增强 NocoBase 的能力。
NocoBase 团队和用户开发者在不断丰富插件的数量，你可以按需选择。

插件 介绍
多数据库数据源 连接第三方数据库，将多个数据库实例作为 NocoBase 的数据源
API 数据源 将 API 作为 NocoBase 的数据源
连接外部数据表 通过 federated/fdw 连接外部数据表
拦截器（工作流） 在触发操作前，进行逻辑校验或者数据预处理，并返回相应的提示
审批（工作流） 工作流审批节点，支持或签、会签等多人审批模式，支持退回、撤回等常见操作
JSON-query 从 JSON 对象读取和处理数据以供继续使用
流程变量（工作流） 在计算流程中对多个节点结果进行计算和拼装，保存为临时值，供全流程使用
组织架构 同步或管理组织架构
审计日志 提供全局和每一条数据的操作日志
嵌入第三方系统 将 NocoBase 嵌入第三方系统
第三方系统嵌入 将第三方系统嵌入 NocoBase
