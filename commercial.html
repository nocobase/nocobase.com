Commercial version

In order to satisfy the advanced needs of our users and to generate revenue for the healthy development of NocoBase, we offer a commercial version based on the open source version. Please note that this is not a separate product from the open source version, it is thanks to the pluginized architecture of NocoBase that the advanced features of the commercial version are all provided as plugins. This means that the open source version will always be our cornerstone and you can use it with confidence.

The commercial version is divided into Licenses, Plugins, and Services sections, and you can choose any section on demand, or you can choose a commercial package that includes all of them.

Licenses

Licenses Description
Commercial Use License Not bound by the AGPL, you can use NocoBase in non-AGPL environments; you can extend your development based on NocoBase without open-sourcing your own source code; you can sell products developed based on NocoBase to others (but not NocoBase itself)
Agent License Sell NocoBase directly as a NocoBase reseller or integrate NocoBase into your products to sell

Plugins

The following commercial plugins are available to greatly enhance the capabilities of NocoBase based on the open source version. NocoBase team and user developers are continually enriching the number of plugins available. You can choose as desired.

Plugin Description
Multiple databases as data source Connect to third-party databases and use multiple database instances as data sources for NocoBase
API as data source Using the API as data source for NocoBase
Connect to external data tables Connect to external data tables via federated/fdw
